# План за: Добави Stripe плащане в eshop-basic

1. Инсталиране на Stripe SDK (npm install stripe)
2. Конфигуриране на Stripe API ключове в .env
3. Създаване на backend маршрут /api/checkout-session за инициализиране на плащания
4. Интегриране на Stripe Checkout бутон във frontend-а (public/index.html)
5. Ръчен тест на процеса на плащане с тестова карта
